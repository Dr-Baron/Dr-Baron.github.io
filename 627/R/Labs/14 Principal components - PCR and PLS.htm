3. Principal Components Regression

> library(pls)

> pcr.fit = pcr( mpg ~ . - name - origin + as.factor(origin), data=Auto )

# Using all variables except name

 

> summary(pcr.fit)

 

 

TRAINING: % variance explained

# The X row shows % of X variation contained in the given number of PCs.

# The mpg row shows R2 (% of Y variation explained) from the PC regression. The usual linear regression on all 8 variables has the same R2 as PCR that uses all 8 principal components.

 

> reg = lm( mpg ~ .-name-origin+as.factor(origin), data=Auto )

> summary(reg)

Multiple R-squared: 0.8242

> pcr.fit = pcr( mpg ~ .-name-origin+as.factor(origin), data=Auto, scale=TRUE, validation="CV" )

# Next, we can fit a model with the desired number of principal components, obtain predicted values, and calculate the prediction mean-squared error. For example:

 

> n = length(mpg);

> Z = sample(n,n/2);

> PLS = plsr( mpg ~ .-name-origin+as.factor(origin), data=Auto[Z,], scale=TRUE, ncomp=6 );

> Yhat = predict( PLS, newdata=Auto[-Z,], ncomp=6 )

> MSE = mean((Yhat - mpg[-Z])^2)

> MSE

[1] 9.619134

> RMSE = sqrt(MSE)

> RMSE

[1] 3.101473
